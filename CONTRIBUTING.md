# Contributing

感谢你改进 Open-Becoming。

## 提交前

1. 从最新默认分支创建独立分支。
2. 保持 `user` 与 `char1`–`char6` 这组公开标识，除非改动包含完整迁移方案。
3. 不要提交真实姓名、对话、截图、设备名、本机绝对路径、数据库、记忆、token 或 API key。
4. 新增图片、字体、图标或代码时，确认拥有再分发权并更新第三方声明。
5. 为行为变化补充或更新测试。

运行检查：

```bash
python -m unittest discover -s tests -v
```

## 许可

提交贡献即表示你有权提交这些内容，并同意以项目当前的 PolyForm Noncommercial 1.0.0 条款许可你的贡献。第三方内容必须保留其原许可和必要声明。
